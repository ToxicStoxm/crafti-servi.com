Eternal SMP

Beschreibung und Informationen:

EternalSMP ist unser aktueller Server der solange aktiev beleiben wird bis alle Spieler das Interesse daran verloren haben. Für Regeln und weitere Informationen zum Server schau bitte in unserem Discord vorbei.

Einstellungen:

  • Gamemode: Survival
  • Difficulty: Hard
  • Seed: Unknown
  • Whitelist: Enforce
  • Online-Mode: True
  • World: Normal
Hinweis: Die Einstellungen in der Konfiguration sind ebenfalls auf Englisch, daher werden sie hier auch auf Englisch angezeigt.
